# 제2회 미래에셋대우 
"금융과 빅데이터의 만남!"

빅데이터라는 용어가 대중적으로 활용되기 전부터 금융 시장에서 쏟아져 나오는 데이터는 빅데이터였습니다.  
특히, 주식 시장에서 매일 나오는 데이터는 오래전부터 분석가들의 도전과제였습니다.  
미래에셋대우는 대한민국의 큰 축을 담당하는 증권사로서 대한민국 금융 시장의 발전과 청년 여러분의 지원을 목적으로   
사회적 소명을 다하기 위해 제2회 미래에셋대우 빅데이터페스티벌을 준비했습니다.

특히, 이번 제2회 미래에셋대우 빅데이터페스티벌은 참여자에게 네이버 테크 에반젤리스트 기술 가이드 및 멘토링을 제공하여   
인공지능 스피커 Clova 음성기반 금융 서비스 개발을 주제로 포함하여 개발자의 꿈을 지닌 분들에게도 참여의 기회를 열었습니다.

금융 시장의 빅데이터를 활용하여 분석을 하고 싶다면?  
현업의 금융 빅데이터 전문가들의 멘토링을 받고 경험을 공유하고 싶다면?

지금 바로 제2회 미래에셋대우 빅데이터페스티벌에 참여해서  
현업의 금융 빅데이터 전문가들의 멘토링을 받고 다양하고 창의적인 방법으로 분석을 해보세요!

증권사 최초로 알고리즘 경진대회를 성대하게 마친 미래에셋대우가 다시한번 대학(원)생 여러분을 찾아왔습니다!    
제1회 때 보다 더 많은 주제, 더 커진 상금과 특전이 여러분을 기다립니다!
