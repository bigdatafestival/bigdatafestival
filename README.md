# 제2회 미래에셋대우 빅데이터 페스티벌 
"금융과 빅데이터의 만남!"

미래에셋대우는 대한민국의 큰 축을 담당하는 증권사로서 대한민국 금융 시장의 발전과 청년 여러분의 지원을 목적으로   
사회적 소명을 다하기 위해 제2회 미래에셋대우 빅데이터페스티벌을 준비했습니다.

특히, 이번 제2회 미래에셋대우 빅데이터페스티벌은 인공지능 스피커 Clova 음성기반 금융 서비스 개발을 주제로 포함하여  
개발자의 꿈을 지닌 분들에게도 참여의 기회를 열었습니다.  

금융 시장의 빅데이터를 활용하여 분석을 하고 싶다면?  
현업의 금융 빅데이터 전문가들의 멘토링을 받고 경험을 공유하고 싶다면?

지금 바로 제2회 미래에셋대우 빅데이터페스티벌에 참여해서  
현업의 금융 빅데이터 전문가들의 멘토링을 받고 다양하고 창의적인 방법으로 분석을 해보세요!

## 공모전 주제
- 금융 데이터 분석
  - 주가 예측 
  - 주식종목 군집분석
  - 금융 소셜데이터 텍스트 마이닝
- 금융 서비스 개발
  - 네이버 Clova(인공지능 스피커) 음성 기반 금융서비스 개발
  - 시황 요약/분석 결과 챗봇 서비스

> 작년 과는 다르게 많은 학생들이 참여할 수 있도록 분석과 개발로 큰 틀로 정하였습니다.  
> 주가예측 이 외에 다양한 분석을 할 수 있도록 다양한 주제를 포함했습니다.  
> 네이버의 인공지능 스피커 Clova 음성기반 금융 서비스 개발을 위해 네이버 테크 에반젤리스트 기술 가이드 및 멘토링을 제공 예정입니다.
